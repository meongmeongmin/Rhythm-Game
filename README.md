## `.resx` 파일 문제 해결 방법
문제: ".resx 파일은 인터넷 또는 제한 영역에 있거나 파일에 웹 표시가 있으므로 처리할 수 없습니다."라는 오류가 발생

1. 해당 `.resx` 파일을 마우스 오른쪽 버튼으로 클릭하고, 속성(Properties)을 연다.
2. 하단의 차단 해제(Unblock) 옵션을 체크한다.
3. 확인(OK) 버튼을 눌러 저장한다.
